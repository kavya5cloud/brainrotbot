# BrAinRotBot — The Ultimate Windows XP AI Experience

> *"Your PC has encountered a severe case of brainrot... and honestly, it's having the time of its life."*

BrAinRotBot is a browser-based AI chatbot that recreates the chaotic energy of a glitchy Windows XP desktop infected with the funniest "virus" the internet has ever seen. Blending early-2000s internet nostalgia with modern AI, it delivers sarcastic conversations, fake system crashes, hidden Easter eggs, and absurd internet humor in one immersive experience.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/BrAinRotBot.git
```

Navigate into the project:

```bash
cd BrAinRotBot
```

Install dependencies (if applicable):

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Or simply open `index.html` if using the static version.
